# HealthyAI Admin (Standalone)

Ung dung admin rieng de mo tab rieng, quan ly bac si va theo doi thong ke cuoc hen.

## Chay local

1. Cai dependency:

```bash
npm install
```

2. Tao file `.env` (co the copy tu `.env.example`):

```env
VITE_API_URL=http://localhost:5001/api
```

3. Chay admin app:

```bash
npm run dev
```

Mac dinh admin app chay o `http://localhost:5174`.

## Yeu cau backend

- Backend phai dang chay.
